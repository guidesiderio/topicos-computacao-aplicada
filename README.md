# Tópicos em Computação Aplicada — UFPI

Repositório com os códigos das tarefas da disciplina Tópicos em Computação Aplicada,
com foco em Blockchain e Finanças Descentralizadas (DeFi).

**Professor:** Glauber Dias Gonçalves  

## Estrutura

| Pasta | Conteúdo |
|-------|----------|
| `tarefa01/` | - |
| `tarefa02/` | Interagindo com a rede P2P via Web3 |
| `tarefa03/` | Coleta e análise de taxas de transações Ethereum (EIP-1559) |

## Configuracao

```bash
pip install -r requirements.txt
```

Crie um arquivo `.env` na raiz do projeto com sua chave da Alchemy:

```
ALCHEMY_API_KEY=sua_chave_aqui
```
